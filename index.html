Welcome to Multivideo

This project is intended to be a semi-universal design base for displaying various video media on a single page.

The basic problem it solves is how to easily access live video streams happening simultaneously. The UI is meant to allow for easy access to streaming "teams" as well as individual streamers. The design could be utilized to display non-live video, but the focus of this project is not exactly aligned to that use.

This project leverages code from a different project (also by @luke80) called jQuery-AnimateReplace. It's implementation here looks something like this:

$(".videotile").click(function(){
	$(this).animateReplace("flip",***SOME_NEW_CONTENT***);
});
